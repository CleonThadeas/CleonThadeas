<summary><b>▶ About Me</b></summary>

<br>

Hello, I'm **Andri Dwi Somantri**, also known as **Cleon Thadeas**.

I'm a developer who is constantly curious about emerging technologies and how they can be transformed into meaningful digital products.

Currently, I work as a **Mobile Developer**, building applications while continuously exploring modern software development practices.

My journey began with **Laravel**, which became the foundation of how I learned web development, software architecture, and backend engineering. Even today, Laravel remains one of my favorite frameworks because it played a significant role in shaping my development mindset.

Over time, I expanded my skill set into **Flutter** for mobile applications, **Nuxt.js** for modern web experiences, and **Golang** for backend services whenever scalability and performance become critical requirements.

Beyond writing code, I enjoy building products, experimenting with ideas, exploring artificial intelligence, and continuously learning new technologies that push me beyond my comfort zone.

My goal is simple:

> Build useful products, solve real problems, and keep growing as a developer.
